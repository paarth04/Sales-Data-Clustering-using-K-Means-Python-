# Sales-Data-Clustering-using-K-Means-Python-
Sales Data Clustering using K-Means (Python)

This project demonstrates how to apply K-Means Clustering on sales data to identify meaningful customer or order patterns using unsupervised machine learning.
We use SALES and QUANTITYORDERED as features, apply feature scaling, determine the optimal number of clusters using the Elbow Method, and finally assign cluster labels to the dataset.
